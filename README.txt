== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin to the `/wp-content/plugins/` directory or install it via the admin panel
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you should see new option in the administration panel, called "Wemovo BT" 
4. Go to "Wemovo BT" and activate your plugin with the public key provided by Wemovo GmbH
5. Place [shortcode] in your pages
